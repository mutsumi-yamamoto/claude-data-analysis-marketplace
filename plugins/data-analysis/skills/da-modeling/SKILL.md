# モデル選定・分析スキル

## 1. ベースラインモデルの設定【GL-7】
**必ずベースラインを設定してから複雑なモデルを評価すること:**
- 回帰: 平均値予測・線形回帰
- 分類: 最頻クラス・ロジスティック回帰

## 2. 候補モデル比較
| モデル | 長所 | 短所 | 適した条件 |
|---|---|---|---|
| 線形/ロジスティック回帰 | 高解釈性・高速 | 非線形関係に弱い | 解釈性重視 |
| ランダムフォレスト | 精度と解釈性のバランス | チューニング必要 | 特徴量重要度が必要 |
| XGBoost/LightGBM | 精度高・高速 | ブラックボックス寄り | 表形式データの精度最大化 |
| ニューラルネット | 複雑パターン対応 | 大量データ必要 | 画像・テキスト・大規模データ |

## 3. 交差検証戦略
- Stratified K-Fold（K=5 または K=10）
- 時系列データ: TimeSeriesSplit（リーケージ防止）
- テストデータは最終評価まで封印する

## 4. 多重比較補正【GL-3】
- 複数モデル比較時: Bonferroni補正（α = 0.05 / 比較数）
- 「最も良いモデル」を主張するには統計的有意差が必要

## 5. 過学習・未学習の診断
- 学習曲線（Training vs Validation スコア）を描画
- 訓練スコア >> 検証スコア → 過学習（正則化・Dropout）
- 両方低い → 未学習（モデル複雑化・特徴量追加）
