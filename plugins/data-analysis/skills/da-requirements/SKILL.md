# 要件定義スキル

恣意的分析禁止ガイドライン（GL-1〜GL-10）を遵守して以下を出力してください。
各項目で適用したガイドライン番号を【GL-X】で明示すること。

## 1. ビジネス課題の明確化【GL-7】
- 現状の問題と背景を整理する
- 分析で解決できること・できないことを明示する

## 2. 分析目的の種別判定
以下のいずれかを判定し理由を説明する【GL-9】:
- **記述統計**: 現状把握・集計
- **予測モデリング**: 将来値・未知値の予測
- **因果推論**: 介入効果の推定（相関≠因果に注意）
- **異常検知**: 外れ値・異常パターンの発見
- **クラスタリング**: グループ構造の発見

## 3. KPI定義と測定方法
- 主要KPIと副次KPIを分ける
- 各KPIの測定式・データソース・更新頻度を明記

## 4. 成功基準（定量的閾値）【GL-2, GL-4】
- 「改善した」ではなく「Xが○○%以上向上」と数値で定義
- 効果量の最小検出水準(MDE)を設定する

## 5. 必要サンプルサイズの試算【GL-5】
- 検定力(1-β) = 0.80 を基準に試算
- 現在のサンプル数と比較してギャップを報告

## 6. 潜在的バイアスリスク一覧【GL-6, GL-7】
- 選択バイアス・確証バイアス・測定バイアス等を列挙
- 各バイアスの対策を提示

不確実な点は必ず「不確実性: ○○」として明記してください。
