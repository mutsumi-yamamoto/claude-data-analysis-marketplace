# 評価・改善スキル

## 1. 評価指標の選択【GL-2】
**複数指標を使用し単一指標最適化を避けること:**
- 分類: Accuracy + F1 + AUC-ROC（クラス不均衡時はF1優先）
- 回帰: RMSE + MAE + R²（外れ値が多い場合はMAE優先）
- ビジネス指標と統計指標の両方を報告する

## 2. 統計的有意性の検定【GL-1, GL-3】
- モデルA vs モデルB: 対応ありt検定（同じCV fold上）
- p値の境界値（p=0.04〜0.06）は「限界的有意」と明記
- Bonferroni補正: α_adjusted = 0.05 / 比較モデル数

## 3. 効果量の評価【GL-2】
- Cohen's d < 0.2: 効果なし（統計的有意でも実務的に無意味）
- Cohen's d 0.2〜0.5: 小効果
- Cohen's d 0.5〜0.8: 中効果
- Cohen's d > 0.8: 大効果

## 4. モデルの公平性チェック
- 人口統計グループ間でのモデル精度のばらつきを確認
- 特定グループに対する偏りがないか分析

## 5. 改善ループ
- エラー分析: 誤分類サンプルのパターンを調べる
- データ拡充: どの種類のデータが最も効果的か
- アーキテクチャ改善: 過学習→正則化、未学習→モデル複雑化
