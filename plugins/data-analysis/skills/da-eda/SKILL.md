# データ前処理・EDAスキル

恣意的分析禁止ガイドライン（GL-1〜GL-10）を遵守して以下を出力してください。

## 1. データ品質チェックリスト
- 行数・列数・データ型の確認
- 欠損値の数・割合（列ごと）
- 重複行の検出
- 値域チェック（負になれない値が負になっていないか等）

## 2. 外れ値検出と処理方針【GL-6】
**客観的基準のみ使用（恣意的除外禁止）:**
- IQR法: Q1 - 1.5×IQR ～ Q3 + 1.5×IQR
- 3σ法: 平均 ± 3標準偏差
- 除外した場合は除外前後の比較を必ず実施【GL-7】

## 3. 記述統計量
- 平均・中央値・標準偏差・IQR
- 歪度・尖度（正規性の確認）

## 4. 変数間の関係【GL-9】
- 散布図 + Pearson/Spearman相関係数
- 「相関がある」止まりで因果を主張しない

## 5. 欠損値の処理方針
- 欠損割合 < 5%: 平均/中央値補完を検討
- 欠損割合 5〜20%: 多重代入法(mice等)を検討
- 欠損割合 > 20%: 変数の除外またはモデルベース補完を検討
- MAR / MCAR / MNAR の区別を必ず確認する
