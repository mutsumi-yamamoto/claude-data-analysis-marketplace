# 管理・運用スキル

## 1. モデル監視指標
- **データドリフト**: 入力特徴量の分布変化（PSI > 0.2で警告）
- **コンセプトドリフト**: 予測精度の低下
- **予測分布のシフト**: 出力値の分布変化

## 2. 再学習トリガーの基準
- 精度が閾値（例: F1 > 0.80）を下回った場合
- データドリフト検知時（PSI > 0.25）
- 定期スケジュール（例: 月次）

## 3. 分析再現性の確保【GL-10】
- 乱数シードを固定（random_state=42等）
- 使用ライブラリのバージョンをrequirements.txtに記録
- 分析手順をJupyter Notebook等で文書化

## 4. バージョン管理方針
- モデルの重み・ハイパーパラメータ・学習データのバージョンを記録
- MLflow / DVC 等のツールの使用を推奨

## 5. 倫理・コンプライアンスチェック
- 個人情報・機密データの取り扱い確認
- モデル出力の説明責任（SHAP等で説明可能にする）
- データ収集・使用の同意確認
