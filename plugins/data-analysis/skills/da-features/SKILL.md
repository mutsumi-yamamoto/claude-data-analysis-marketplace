# 特徴量エンジニアリングスキル

## 1. 特徴量候補一覧と選定根拠
- ドメイン知識に基づく特徴量
- 統計的特徴量（移動平均・ラグ特徴量等）
- 交互作用特徴量（掛け算・比率）

## 2. データリーケージ防止策【GL-10】
- 目的変数と直接関連する特徴量を除外
- 時系列の場合: 未来の情報が混入していないか確認

## 3. 多重共線性チェック【GL-9】
- VIF（分散拡大係数）> 10 の特徴量を要確認
- 相関行列で r > 0.9 のペアを特定

## 4. 特徴量重要度の評価
- Permutation Importance（モデル非依存）
- SHAP値（加算的・解釈可能）を優先して使用
- Gini重要度はバイアスがあるため使用しない

## 5. スケーリング方針
- 標準化（StandardScaler）: 正規分布に近いデータ
- 正規化（MinMaxScaler）: 値域を揃えたい場合
- ロバストスケーリング（RobustScaler）: 外れ値がある場合
